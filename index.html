In-Hand Object Rotation via Rapid Motor Adaptation
Anonymous CoRL Submission
Paper ID: 352

Abstract

Generalized in-hand manipulation has long been an unsolved challenge of robotics. As a small step towards this grand goal, we demonstrate how to design and learn a simple adaptive controller to achieve dynamic in-hand object rotation using only finger tips. The controller is trained entirely in simulation on only cylindrical objects, which then -- without any fine-tuning -- can be directly deployed to a real robot hand to rotate dozens of objects with very different object properties, including sizes, shapes, and weights. This is achieved via rapid online estimation and adaptation of the robot's controller to the object properties, using only proprioception history, without any visual or tactile sensing. Furthermore, natural dynamic gaits of fingers automatically emerge from training the control policy via reinforcement learning, without the need for any heuristic rewards or pre-programming for gaits.
